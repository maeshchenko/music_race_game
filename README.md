# 2107 — ритм-гонка

**СДВГ-таймкиллер на максимум дофамина.** Цель — не возврат на следующий день, а
ОДНА сессия максимальной длины: чтобы можно было залипнуть на часы безвылазно,
не считая время. Весь дизайн заточен под дофамин/окситоцин/эндорфин и
непрерывный поток без точек выхода.

Машина (ВАЗ-2107) едет по бесконечной вечерней провинции, собирая неоновые
блоки. Блоки садятся на РИТМ-секцию трека (бочка/снейр/бас на доли) — сбор
бьёт в долю, как ударные; ведущая мелодия вплетается редким акцентом. Звук
сбора квантуется в строй песни (модель Beat Saber / Audiosurf).

## Видение (исходный бриф)

Игра для СДВГ-людей. Цель — не возврат на следующий день, а одна сессия
максимальной длины: идеальный таймкиллер, чтобы залипнуть на часы безвылазно,
не считая время.

Основные тезисы при разработке:

1. Посмотреть, что уже есть, и как улучшить дофаминовую петлю — максимум
   дофамина + окситоцина + эндорфина, чтобы можно было играть максимально долго.
2. ориентиры: Beat Saber / Audiosurf
3. Это ритм-игра, и каждый блок должен ощущаться как «ты делаешь музыку»,
   вызывать «купание мозга в дофамине».

## Ключевые механики

- **Бесконечный DJ-сет** — треки склеиваются бесшовно (бит-сшивка, непрерывные
  дорога/скорость/музыка), без экрана результатов и без ожидания. Финиш трека =
  салют + камео камеры, но поток не рвётся.
- **Комбо → fever I/II/III** + комбо-кольцо у машины; музыка обрастает слоями.
- **⚡ Турбо «Ускорение»** — копишь полный бак, жмёшь ЛКМ: реальный разгон
  темпа+мира, машина-поезд (таран преград без штрафа), скоростной блюр.
- **Паттерны блоков → дофамин-предсказание** — волна/зигзаг/лесенка/маятник/
  стена/дабл, выбор по энергии секции; перед музыкальным дропом — разгон
  (маятник), НА дропе — burst + награда (МЕГА/золото). Читаемые фигуры дают
  дофамин от предсказания, не чистый рандом.
- **Здоровье мелодии** — пропускаешь блоки → ведущая мелодия трека гаснет
  (~1.3с до тишины); любой сбор мгновенно возвращает её на полную. Игра
  «стартует» с первого собранного блока (до него мелодия не трогается).
- **Разогрев сессии** — чем дольше играешь непрерывно, тем выше множитель и
  богаче картинка (сброс только при выходе).
- **Variable-ratio сюрпризы** — золотые волны, mystery-блоки (вкл. ~2% легенду),
  near-miss бонусы, микро-цели каждые 10–20с.
- **Звук всегда в ключе трека** — бочка/снейр/нота-дубль/искра квантуются в
  тональность+лад песни (нельзя «сыграть фальшиво»).

## Реализовано (ключевые системы)

- **Бесшовный бесконечный режим** — один непрерывный `Tone.Transport`, цепочка
  трек-сегментов (`conductor.ts` + `game/chain.ts`), сваренных встык; дорога —
  одна глобальная гладкая лента (`game/road.ts`), без шва. Скорость сварена у
  концов (нет толчка), клок на тиках (инвариант к темпу).
- **Финиш-климакс на стыке** — bullet-time (темп → 0.4× и выстрел назад),
  полноэкранная вспышка + ударная волна + слэм-текст «УРОВЕНЬ ПРОЙДЕН» + салют.
- **Турбо «Ускорение»** — копишь полный бак → ЛКМ: РЕАЛЬНЫЙ разгон темпа
  (через `bpm`, питч не плывёт), машина-поезд (таран преград), скоростной блюр.
- **Дофамин-петля** — комбо→fever I/II/III + комбо-кольцо, послойная музыка,
  золотые волны, mystery (вкл. ~2% легенда), near-miss, perfect-streak,
  микро-цели, разогрев сессии (множитель растёт всю сессию), 5 пикапов.
- **Ритм-транс (#эндорфин)** — камера/окна/фонари/блум пульсируют по биту;
  directional hit-feedback; цветовая эскалация по комбо; haptic.
- **Анти-скука** — новизна каждые ~40с: смена темы/погоды + **биом-сета**
  (спальник/центр/промзона/окраина), въезжаешь в новый район плавно.
- **Генерация блоков (`game/blocks.ts`)** — поток ведёт РИТМ (kick/snare/bass на
  доли), lead-соло — редкий акцент в среднем регистре (раньше доминировало →
  высокие «пищащие» блоки). 6 паттернов полос (волна/зигзаг/лесенка/маятник/
  стена/дабл) по энергии секции; детект build-up→drop из `activityPerBar`
  (`rhythm.ts`): маятник-разгон перед дропом, burst + МЕГА/золото НА дропе.
  Блоки единого размера (без МЕГА-раздувания). Достижимость ±1 — твёрдый пол.
- **Звук сбора в строй трека (`game/sfx.ts`)** — все тональные слои (lead/бас/
  искра) квантуются в `key.tonic`+`mode` песни (`setKey`/`snap`); строй едет за
  тональностью цепочки на стыках. «Искра» = класс высоты собранной ноты в звонкой
  октаве. Нет нот вне ключа.
- **Здоровье мелодии (`game.ts`)** — `leadHealth` глушит лид-трек кондуктора
  (`setLeadGuide`): промах взводит спад до тишины за ~1.3с, сбор мгновенно
  возвращает полную; до первого сбора мелодия не трогается.
- **Мир** — 7 архетипов зданий, неон-билборды, эстакады.
- **Перф/звук** — кэш reverb-IR, JIT-стемы (≤2 ансамбля разом), увеличенный
  lookAhead, анти-клип дак на стыке. Диагност-perf-логи в коде (НЕ удалять).

## Управление

- мышь — руль
- ЛКМ (или Shift/E) — турбо (когда бак полон)
- пробел — пауза · C — вид из салона · F — полный экран · Esc — меню

## Технологии

TypeScript + Vite, Three.js (рендер + bloom), Tone.js (живой синтез на одном
непрерывном транспорте), процедурный MIDI (`midi-gen`).

## Заметки для агентов

- **Perf-логи** (`[longtask] [build] [append] [jank] [finale] [seam-work]`,
  помечены `НЕ УДАЛЯТЬ`) — живые метрики оптимизации, не стирать без явной просьбы.
- **Жанры музыки** (`grimerun/outrun/eurobeat`) не менять — другие не подходят.
- **Блоки = РИТМ, не lead-соло.** Решение сессии: lead-соло доминировал → блоки
  были высокие/«пищащие» и не в ритм. Теперь поток ведёт ритм-секция, lead —
  редкий акцент (`blocks.ts`: вес lead 20, не 100; lead только на сильную долю с
  кулдауном, опущен в средний регистр). НЕ откатывать к lead-доминанте, даже если
  исходный бриф говорит «играешь мелодию» — игрок явно попросил ритм.
- **Звук сбора квантуется в строй** (`sfx.ts setKey/snap`) — не убирать: иначе
  перк-слоты (дефолт pitch=60) звенят C вне тональности = фальшь.
- **Кап-плотности держит РАННЕЕ (на-бите) время** (`blocks.ts`: `{...c, t: last.t}`)
  — не возвращать время победителя, иначе блок уезжает позже доли = отставание ритма.
- **Здоровье мелодии**: восстановление мгновенное (`leadHealth=1` на сбор), спад
  ~1.3с (`dt/1.3`), включается с первого сбора (`engaged`). Параметры подобраны
  игроком — менять только по запросу.
- **Турбо называется «УСКОРЕНИЕ»** (бывш. «Закись Азота») — строки в `game.ts`.
