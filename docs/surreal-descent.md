# Сюрреалистический спуск (Surreal Descent)

> ⚠️ **Этот документ описывает СТАРУЮ 6-фазную арку (лес/бэкрумсы/луна/поле).**
> Актуальная арка переписана: гонка → краш → лес → роща → озеро (лиричный
> Firewatch-финал). Источник истины по сюжету — `surreal-descent-scenario.md`.
> Раздел ниже сохранён как история и как описание пропсетов (бэкрумсы/луна/поле
> остались в коде world.ts, но убраны из основного пути).

> «Игра тихо сходит с ума». Заезд начинается обычной аудиосёрф-гонкой, а по
> дистанции мир сам уезжает в сюр — в духе *The Stanley Parable*: начинается
> мундан, ломает ожидания, дальше странность нарастает.

## Зачем

Хотели не отдельный «уровень», а **подкрадывающуюся** трансформацию: игрок
ничего не выбирал — мир сам поехал. Дуга фаз:

```
гонка → лес → выход из машины, бежишь пешком → бэкрумсы → луна → сюр-поле с грибами
```

Текстовый нарратор (суб-полоса снизу) комментирует спуск и в финале ломает
4-ю стену.

## Как устроено

### Ось — дистанция, не время
Фазы стартуют по метрам вдоль трассы (`startDist`), а не по секундам. Дистанция
монотонна, переживает турбо и паузу, и уже драйвит геометрию дороги. Спуск
включается **сам** в обычном бесконечном заезде: фаза `race` молчит до порога
(~4000 м), потом Story берёт руль и глушит обычную «новизну тем».

### Стейт-машина — `src/game/story.ts`
`PHASES: PhaseDef[]` — для каждой фазы: `startDist`, кросс-фейд `blendIn`, тема
палитры, `propSet` (словарь декора), `insanity` (0..1), `audioRate` (темп),
`camera`-режим и реплики нарратора. `Story.tick(dist)` бинарным поиском находит
активную фазу, лерпит палитру на входе, отдаёт снимок `StorySnap` — Game
применяет его к миру/звуку/нарратору.

### Точки подключения в `src/game/game.ts`
Один блок в `tick()` после расчёта `dist`:
- `world.setPropSet(...)` при смене словаря декора (разовая пересборка чанков),
- `world.applyThemeColors(...)` — дешёвый покадровый своп палитры/тумана (без
  пересборки осадков),
- `director.setInsanity(...)` — цель глитч-слоя,
- `conductor.setRate(...)` — замедление музыки,
- `narrator.say(...)` — реплики по дистанции.

Обычная «новизна тем» (`game.ts`, ~919) глушится через `if (!this.story?.active)`.

### «Безумие» — `src/game/intensity.ts` + `src/game/glitchPass.ts`
Один скаляр `IntensityDirector.insanity` 0..1 (медленный лерп к цели фазы) гонит
несколько дешёвых эффектов:
- крен/тилт камеры (`game.ts`, после FOV-блока),
- глитч-шейдер `glitchPass.ts` (RGB-сдвиг/хром. аберрация + scanline-jitter +
  редкие инверт-вспышки), один `ShaderPass` между bloom и OutputPass; гонится
  юниформом `uInsanity`. На мобайле composer'а нет → fallback CSS-фильтр на
  канвасе (`hue-rotate/saturate/contrast`).

### Нарратор — `src/game/narrator.ts`
Один DOM-элемент в нижней трети: typewriter-раскрытие по букве, держится, гаснет,
очередь реплик. Только текст, ноль аудио-ассетов. Реплики авторятся в `story.ts`
по фазам (по-русски).

### Пропсет-слой — `src/game/world.ts`
`PropSetId = 'province' | 'forest' | 'backrooms'`. `setPropSet(id)` перекрашивает
полотно (асфальт→земля/ковёр) и пересобирает чанки (`rebuild()`); `writeChunk`
диспатчит по флагам `forest`/`backrooms`/`bare`. Все пулы инстансов аллоцируются
**один раз** в конструкторе — пропсет лишь меняет, какие пулы заполняются vs
прячутся. Zero-alloc recycling цел.

- **Лес:** дома/гаражи/киоски прочь; плотная многорядная стена высоких ёлок
  вплотную к кромке (ёмкость пулов деревьев поднята 12→96); асфальт→грунтовка,
  разметки нет; фонари → **светящиеся грибы** (`pMushStem` + `pMushCap`,
  неоновая шляпка, мягкое пятно света).
- **Бэкрумсы:** жёлтые обои-стены по бокам (ширина комнаты варьируется по
  детерм. хешу — разные планировки), потолочные плиты с **люминесцентными
  лампами** «как попало» (`pWall`/`pCeil`/`pFluoro`), поперечные
  **перегородки-проёмы** (едешь из комнаты в комнату), пол → сырой
  грязно-горчичный ковёр. Эстетика по Backrooms Level 0: моно-жёлтые потёртые
  обои, сырой ковёр, гудящие лампы, лабиринт пустых комнат, изоляция.

### Пеший режим — `src/game/runner.ts`
Бегущий человечек: голова, тело, две руки, две ноги. Руки/ноги висят на
пивот-группах (плечо/бедро), качаются вокруг оси X. **Фаза шага = `dist·k`** —
шаги совпадают с прокруткой дороги, амплитуда/каденс растут со скоростью; руки и
ноги в противофазе; тело подпрыгивает. Управление то же (рулёжка полосами) —
**визуальный рескин**, не новая механика. Клавиша **P** прячет машину и
показывает человечка; на фазе `onfoot` включается сам.

### Звук
`conductor.setRate(rate, dur)` рампит темп (замедление по фазам). Защищён от
гонки рампов (`lastRampT` + try/catch) — два rampTo на один аудио-момент бросали
`Start time must be strictly greater…` (Tone).

| Что | Как |

| Пеший тумблер | `P` |

## Статус (2026-06-16)

**Вся дуга готова и проверена в браузере (ноль ошибок консоли):**
- Атмосфера: своп темы/тумана/цвета, insanity → глитч + крен камеры, замедление
  музыки, нарратор.
- Лес: чаща + грибы-фонари + грунтовка.
- Пешком: бегущий человечек с анимацией конечностей (клавиша P + авто на onfoot).
- Бэкрумсы: комнаты + стены/потолок/лампы + проёмы + ковёр.
- Луна: звёздное небо (precip `'stars'`), лунный грунт + серые камни, чёрная
  пустота.
- Сюр-поле: парящие светящиеся грибы-гиганты, пастельные орбы, «странные штуки»
  (будки в неверном масштабе/вверх ногами), пастельная трава.
- Сценарий: связная дуга нарратора (см. `surreal-descent-scenario.md`), реплики
  по фазам в `story.ts`.

**Осталось (полировка, опционально):**
- Lift-off на луне (вертикальный отрыв аватара от дороги, полотно гаснет) — пока
  едешь по лунному грунту, без буквального отрыва.
- foot-камера ниже к высоте головы + покачивание бега (сейчас обычный chase).
- Сужение трассы (`wideAt`) для леса/коридоров.
- Внешние CC0-GLB вместо процедурных ёлок/человечка/грибов.
- Финальная петля: после поля мягко вернуть в гонку («…как всегда»).
- Per-frame bob/вращение парящих предметов поля (сейчас статично подвешены).
